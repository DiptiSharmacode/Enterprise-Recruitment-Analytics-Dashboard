## Project Overview

The Enterprise Recruitment Analytics Dashboard is an end-to-end HR Analytics project designed to analyze recruitment performance, hiring efficiency, candidate quality, and recruitment source effectiveness.

The project demonstrates a complete analytics workflow from data warehousing to interactive dashboard development using SQL, Python, and Power BI.

## Business Problem

Recruitment teams often struggle to answer questions such as:

- Which recruitment sources produce the highest-quality candidates?
- What is the overall hiring rate?
- How long does hiring take?
- Which recruitment channels have the best offer acceptance rate?
- Where are candidates dropping off in the recruitment funnel?

This dashboard helps HR leaders make data-driven recruitment decisions.
